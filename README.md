## Jimmy-Server
> 用户快速搭建一个基于Node.JS的KOA框架的一个后端服务脚手架

[官方文档](http://www.baidu.com) - 'http://www.baidu.com'

### 安装依赖

```
npm install
```

### 开发环境运行

```
npm run dev
```

### 生产环境运行

- 查看服务器运行pm2列表

  ```
  npm run list
  ```

- 开启服务

  ```
  npm run pro
  ```

- 停止服务

  ```
  npm run stop
  ```
